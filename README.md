# Formulário Simples em Java Swing

Projeto desenvolvido para praticar conceitos básicos de interfaces gráficas em Java utilizando Swing.

## Funcionalidades

- Cadastro de Nome
- Cadastro de Idade
- Cadastro de CPF
- Botão Enviar
- Botão Limpar Campos
- Exibição dos dados digitados

## Tecnologias Utilizadas

- Java
- Java Swing
- AWT Event Handling

## Conceitos Praticados

- JFrame
- JLabel
- JTextField
- JTextArea
- JButton
- ActionListener
- Layouts (FlowLayout)
- Eventos em interfaces gráficas

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Abra o projeto em sua IDE favorita.

3. Execute a classe:

```bash
SinplesGuir.java
```

## Objetivo

Este projeto foi criado com fins de aprendizado para compreender os fundamentos do desenvolvimento de interfaces gráficas em Java utilizando Swing.
